ドロップシャドウエフェクト

何かシルエットが出るエフェクトです
合成、オサレ演出等にご使用ください

使い方
アクセサリのXY座標でズレ、Z座標で距離
Siでテクスチャループ数、Trでぼかしの設定を行います。
ポストエフェクトのように見えて実は違うタイプのエフェクトなので、
描画順をモデルより前に指定しておいて下さい。

エフェクト割当->DropShadowRTより、ドロップシャドウを生成したくないオブジェを非表示にする事で特定のオブジェのみ影を生成する事ができます。

重要！！
どうもMMEのバージョンによって、↑の動作を行っても非表示にならないことがありました。
その場合、MMDのウィンドウを最大化、通常化する事で解決したりしなかったりしました。


影色についてはテクスチャを直接編集して下さい
